# Chrome Extension: HLSPlayer
Use hls.js to play HLS(m3u8) in Chrome.  
And it will be more easier to get HLS url with [VideoGrab](https://github.com/sonichy/VideoGrab) extension.

![alt](preview.png)  

|Key|Function|
|:-:|:-:|
|←|Seek Backward|
|→|Seek Forward|
|↑|Volume Up|
|↓|Volume Down|
|M|Mute|
|F|Enter Fullscreen|
|Space|Pause or Play|
|Mouse Left Click|Pause or Play|
